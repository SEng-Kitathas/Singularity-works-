# CIL Shadow Research Crosswalk — 2026-03-28a

This crosswalk translates the main research seams into build pressure for CIL.

## Seam 1 — Complementary Learning Systems / Stability-Plasticity
**Representative seam**: fast plastic memory + slower semantic consolidation + replay/sleep/inhibition.

**Key takeaway**
CIL’s long-running episodic/semantic split is on the right track, but continual systems need explicit replay, slower consolidation, and anti-runaway stabilization.

**Architectural implication**
Promote a first-class volatile session buffer and replay/consolidation engine above/between EPMEM and SMEM.

**Cross-links**
- Contradiction memory
- Promotion gates
- Retrieval compiler

## Seam 2 — Explicit Agent Memory Architecture

**Key takeaway**

**Architectural implication**
